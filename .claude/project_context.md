# Soluciones Ganadoras - Context del Proyecto

## Descripción General
Soluciones Ganadoras es una landing page para servicios financieros que incluye precalificación de crédito, financiamiento e inversiones.

## Tecnología
- **Frontend**: HTML, CSS, JavaScript puro
- **Build Tool**: Vite
- **Node.js**: Para desarrollo y build

## Estructura del Proyecto

### Páginas Principales
- `index.html` - Página principal/landing page
- `precalificacion.html` - Formulario de precalificación de crédito
- `financiamiento.html` - Información y formulario de financiamiento
- `inversiones.html` - Información sobre inversiones

### Scripts
- `precalificacion.js` - Lógica del formulario de precalificación
- `financing.js` - Lógica del módulo de financiamiento

### Estilos
- `styles.css` - Estilos globales del sitio

### Recursos
- `logoSolucionGanadora.jpg` - Logo principal
- `logosolucion.jpg` - Logo alternativo

## Scripts Disponibles
- `npm run dev` - Servidor de desarrollo con Vite
- `npm run build` - Build de producción
- `npm run preview` - Preview del build de producción

## Información Git
- **Rama Principal**: main
- **Rama Actual**: Stiven
- **Último Commit**: Actualziacion Financiamiento

## Notas
- El proyecto usa Vite como bundler para desarrollo
- Es un sitio estático con JavaScript vanilla
- Enfocado en servicios financieros (crédito, financiamiento, inversiones)
