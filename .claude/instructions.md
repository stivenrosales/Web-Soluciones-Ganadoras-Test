# Instrucciones para Claude - Soluciones Ganadoras

## Contexto del Proyecto
Este es un sitio web de servicios financieros llamado "Soluciones Ganadoras" que ofrece:
1. Precalificación de crédito
2. Servicios de financiamiento
3. Información sobre inversiones

## Stack Tecnológico
- HTML, CSS y JavaScript vanilla
- Vite como herramienta de desarrollo
- Sin frameworks frontend (React, Vue, etc.)
- Estructura de proyecto plana (sin carpetas src/)

## Al Trabajar en Este Proyecto

### Estructura de Archivos
- Los archivos HTML están en la raíz del proyecto
- Los archivos JS están en la raíz del proyecto
- Los estilos están en `styles.css` en la raíz
- Las imágenes están en la raíz

### Estilo de Código
- JavaScript: ES6+ moderno
- CSS: Sin preprocesadores, CSS puro
- HTML: HTML5 semántico

### Testing y Preview
- Usar `npm run dev` para probar cambios localmente
- El servidor de desarrollo de Vite recargará automáticamente los cambios

### Git Workflow
- Rama principal: `main`
- Rama de trabajo actual: `Stiven`
- Hacer commits descriptivos en español

## Funcionalidades Principales

### Precalificación
- Formulario interactivo en `precalificacion.html`
- Lógica en `precalificacion.js`

### Financiamiento
- Información y formulario en `financiamiento.html`
- Lógica en `financing.js`

### Inversiones
- Página informativa en `inversiones.html`

## Consideraciones Importantes
- Mantener consistencia visual entre páginas
- Asegurar responsividad en todos los dispositivos
- Validar formularios del lado del cliente
- Mantener el código simple y legible
