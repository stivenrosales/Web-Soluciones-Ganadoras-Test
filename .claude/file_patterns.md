# Patrones de Archivos - Soluciones Ganadoras

## Archivos HTML
Todas las páginas HTML siguen un patrón similar:
- Header con navegación
- Secciones de contenido
- Footer
- Referencias a `styles.css` para estilos
- Scripts específicos de cada página

## Archivos JavaScript
- `precalificacion.js` - Maneja el formulario de precalificación
- `financing.js` - Maneja la lógica de financiamiento

## Estructura de Estilos
`styles.css` contiene todos los estilos globales del proyecto.

## Convenciones
- Páginas HTML en la raíz del proyecto
- Scripts JavaScript en la raíz
- Imágenes/logos en la raíz
- Sin carpeta `src/` - estructura plana
